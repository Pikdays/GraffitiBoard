# GraffitiBoard -- 涂鸦板

## **中文**

### 简介

此项目来源于新东方教育科技集团优播课APP教师端手绘功能。
感谢GitHub上的`Graffiti`,`Draw`以及相关绘画的开源作者。

### 使用

```objc

[self.navigationController pushViewController:[[GraffitiBoardViewController alloc] init] animated:YES];

```
